\subsection{BFS}
Computes $d$, an array of distance from start vertex $v$.
$d[v]=0$, $d[u]=\infty$ if $u$ not connected to $v$. If $(u, w)\in E$ and $d[u]$ known and $d[w]$ unknown, $d[w] = d[u]+1$.\\
